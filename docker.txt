 
- Como procurar imagem no docker hub?

docker search <KEY>
- Como listar imagens locais?
docker image ls

- Como remover uma imagem local?
docker image rm <NAME | ID>

- Como criar um container?
docker run [--name NAME] <IMAGE_NAME>

- Como personalizar sua própria imagem?
a) Criar um arquivo Dockerfile com as definições
b) docker build -t <IMAGE_NAME |IMAGE_NAME:TAG> .

- Como criar um container à partir de sua imagem personalizada?
docker run [-d -p HOST_PORT:CONTAINER_PORT --name NAME] <IMAGE>

- Como listar os containers?
docker ps [-a]
- Como parar um container em execução?
docker [container] stop <ID | NAME> 

- Como iniciar um container parado?
docker [container] start <NAME | ID>

- Como remover um conteiner?
docker [container] rm <ID | NAME>
